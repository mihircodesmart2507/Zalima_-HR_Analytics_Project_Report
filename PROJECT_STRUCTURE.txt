TalentView HR Dashboard - Project Structure
============================================

├── pages/                     # Dashboard pages

├── QUICK_START.md             # Quick start guide
├── PROJECT_STRUCTURE.txt      # This file
└── .gitignore                 # Git ignore file

ARCHIVE/                        # Old pricing analysis files (if any)

HOW TO RUN:
===========
1. Install dependencies: pip install -r requirements.txt
2. Run dashboard: streamlit run app.py
3. Or use: python run_dashboard.py

DATA REQUIREMENTS:
==================
- CSV file with HR employee data
- Required columns: EmployeeNumber, Attrition, Department, JobRole, etc.
- See README.md for full column list
